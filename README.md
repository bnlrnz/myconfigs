# myconfigs installation:

- ```cd ~/.config/```
- ```git clone --no-checkout git@github.com:bnlrnz/myconfigs.git```
- ```cd myconfigs```
- ```git config core.worktree "../../"```
- ```git reset --hard origin/main```
